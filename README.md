# Resumo dos Exercícios de Revisão em Python

Este notebook contém uma série de exercícios práticos para revisar os conceitos fundamentais da linguagem Python, abrangendo operações básicas, variáveis, operadores lógicos, manipulação de entrada/saída (`print` e `input`), e os diferentes tipos de dados (int, float, boolean, string, list, tuple, set, dict).

## Tópico: Operações Básicas e Variáveis/Objetos

Nesta seção, os exercícios focam na declaração de variáveis e na realização de operações aritméticas para cálculos financeiros e contábeis simplificados.

*   **Exercício 1:** Soma de salário e bônus.
*   **Exercício 2:** Cálculo de adicional e salário final com base em um percentual.
*   **Exercício 3:** Determinação do custo total de departamentos, incluindo uma taxa de manutenção.
*   **Exercício 4:** Cálculo do lucro operacional anual, considerando receitas e despesas mensais e anuais.
*   **Exercício 5:** Determinação do EBITDA trimestral, utilizando diversas métricas financeiras.

## Tópico: Operadores Lógicos (<, >, <=, >=, ==, !=, and, or, not)

Esta parte aborda o uso de operadores lógicos e relacionais para criar expressões booleanas que avaliam condições.

*   **Exercício 1:** Verificação da idade mínima para dirigir.
*   **Exercício 2:** Checagem se uma pontuação de crédito está dentro de uma faixa específica.
*   **Exercício 3:** Critérios de aprovação de um aluno, baseados na média e notas individuais.
*   **Exercício 4:** Lógica de elegibilidade para desconto em uma loja, considerando valor de compra e status VIP.
*   **Exercício 5:** Avaliação de condições para aprovação de crédito em uma instituição, utilizando dados de um dicionário `cliente`.

## Tópico: print (básico e f-string) e input

Os exercícios aqui exploram como interagir com o usuário e formatar a saída de informações.

*   **Exercício 1:** Impressão de uma frase simples.
*   **Exercício 2:** Uso de f-strings para exibir mensagens formatadas com variáveis.
*   **Exercício 3:** Coleta de entrada do usuário (`input`) e exibição de uma mensagem de boas-vindas.
*   **Exercício 4:** Solicitação de dois números inteiros, cálculo da soma e impressão do resultado formatado.
*   **Exercício 5:** Leitura de múltiplas notas em uma única linha, cálculo da média e determinação de aprovação, com saída formatada.

## Tópico: Tipos (int, float, boolean, string, list, tuple, set, dict)

Esta seção se dedica à criação e manipulação dos tipos de dados fundamentais em Python.

*   **Exercício 1:** Criação de variáveis de diferentes tipos básicos e verificação de seus tipos.
*   **Exercício 2:** Cálculo do total e da média de vendas a partir de uma lista.
*   **Exercício 3:** Conversão entre tuplas e listas, adição de elementos, ordenação e conversão de volta para tupla.
*   **Exercício 4:** Identificação de itens únicos em uma lista e ordenação alfabética, utilizando conjuntos (`set`).
*   **Exercício 5:** Manipulação de um dicionário de vendas: adição de novos dados, cálculo do total anual, identificação dos meses com maior e menor venda, e extração de chaves e valores para listas.
